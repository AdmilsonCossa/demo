首页简介

首先在index.html中引入如下javascript。 此处使用了requireJS,有兴趣可以学习一下。 大概意思是动态加载main.js。 

<script type="text/javascript" data-main="main" src="assets/require.js"></script>
define(['app' ], function(App) {
	var router = new App.router;
	Backbone.history.start();  
	return App;
});
